# Быстрый старт - TickTick iOS App

## Минимальные шаги для запуска:

1. **Откройте Xcode** → `File` → `New` → `Project`

2. **Создайте iOS App:**
   - Product Name: `TickTick`
   - Interface: **SwiftUI**
   - Language: **Swift**

3. **Добавьте файлы в проект:**
   - Скопируйте все файлы из этой папки в папку проекта Xcode
   - В Xcode: правой кнопкой → `Add Files to "TickTick"...`
   - Выберите все файлы и папки
   - ✅ Поставьте галочку "Copy items if needed"

4. **Замените автоматически созданные файлы:**
   - Замените `TickTickApp.swift` нашим файлом
   - Удалите автоматически созданный `ContentView.swift`

5. **Запустите:**
   - Выберите симулятор (iPhone 15 Pro)
   - Нажмите ▶️ или `Cmd + R`

## Требования:
- Xcode 14.0+
- iOS 15.0+ (в настройках проекта)

## Если что-то не работает:
Смотрите подробную инструкцию в файле `ИНСТРУКЦИЯ_ПО_ЗАПУСКУ.md`
